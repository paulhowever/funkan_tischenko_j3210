# case_4

Реализация функциональной регрессии через линейные функционалы.

- Условие: `../docs/case_4/кейс 4.pdf`
- Основной код: `../src/case_4`
- Тесты: `../tests/case_4`
- Ноутбуки: `../notebooks/case_4`

## Что уже есть

- генерация синтетических функциональных данных на равномерной сетке
- базисы: кусочно-индикаторный, полиномиальный, тригонометрический
- вычисление признаков через численное интегрирование
- OLS и Ridge в матричной форме
- метрики `MSE`, `RMSE`, `R2`
- восстановление весовой функции `w(t)`
