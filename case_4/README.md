# case_4

Реализация функциональной регрессии через линейные функционалы.

- Условие: `../docs/case_4/кейс 4.pdf`
- Основной код: `../src/case_4`
- Тесты: `../tests/case_4`
- Ноутбуки: `../notebooks/case_4`

## Что уже есть

- генерация синтетических функциональных данных на равномерной сетке
- генерация кусочно-гладких данных с локальными особенностями
- базисы: кусочно-индикаторный, полиномиальный, тригонометрический
- вычисление признаков через численное интегрирование
- OLS и Ridge в матричной форме
- метрики `MSE`, `RMSE`, `R2`
- восстановление весовой функции `w(t)`
- исследования: sweep по `m`, sweep по `lambda`, устойчивость к шуму и к сетке

## Где смотреть результаты

- Ноутбук отчета: `../notebooks/case_4/report_case_4.ipynb`

## Что проверяется по заданию

- построение матрицы признаков `Z` через функционалы;
- сравнение минимум двух наборов функционалов;
- влияние числа функционалов `m`;
- влияние регуляризации Ridge;
- устойчивость к шуму и дискретизации;
- интерпретация через восстановленную `w(t)`.
